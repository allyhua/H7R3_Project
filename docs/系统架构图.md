## 7. 中断优先级配置

```
中断优先级配置 (数值越小优先级越高):

优先级 0: DCMIPP帧中断 (最高)
优先级 1: DMA2D传输完成中断
优先级 2: DMA传输完成中断
优先级 3: 定时器中断 (LVGL刷新)
优先级 4: UART串口中断
优先级 5: 触摸屏中断
优先级 6: 外部中断 (按键)
优先级 7: 系统滴答定时器
```
